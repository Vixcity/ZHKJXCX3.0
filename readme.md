## 简要说明

苹果手机的new Date(time) 不支持 'YYYY-MM-DD' 形式传入的参数，也就是说不能解析 YYYY-MM-DD 形式的日期格式
支持 YYYY/MM/DD 形式的日期格式，也支持时间戳
如果数据返回的是 2022-07-19T06:33:50.000000Z 这种格式，
我们需要提取他的日期和时间在进行解析
